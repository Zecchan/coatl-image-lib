SQLite Database
Relative to the ui/ folder, or an absolute path (e.g. D:/coatl/coatl.db).
SQLite is bundled with Node.js via better-sqlite3 — no separate installation needed. The database file is created automatically on first use.
Qdrant Vector Database
Qdrant must be installed and running separately. Download from qdrant.tech or run via Docker:
docker run -p 6333:6333 qdrant/qdrant
